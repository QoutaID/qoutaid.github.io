# Frequently Asked Questions

Pertanyaan yang sering diajukan tentang QoutaID.

{: .fs-6 .fw-300 }

---

## Umum

### Apa itu QoutaID?

QoutaID adalah platform yang menyediakan berbagai layanan dan tools untuk pengembangan digital, termasuk hosting, API services, dan development tools.

### Bagaimana cara mendaftar?

Kunjungi [qouta.id](https://qouta.id) dan klik tombol "Daftar" untuk membuat akun baru.

---

## Akun

### Bagaimana cara reset password?

1. Kunjungi halaman login
2. Klik "Lupa Password"
3. Masukkan email yang terdaftar
4. Cek email untuk link reset password

### Apakah bisa menggunakan multiple devices?

Ya, kamu bisa login di beberapa device sekaligus. Semua sesi akan tersinkronisasi.

---

## Billing

### Metode pembayaran apa yang diterima?

Kami menerima berbagai metode pembayaran:

- Transfer Bank (BCA, Mandiri, BNI, BRI)
- E-Wallet (GoPay, OVO, Dana)
- Kartu Kredit/Debit (Visa, Mastercard)

### Bagaimana cara upgrade paket?

1. Login ke dashboard
2. Kunjungi halaman "Billing"
3. Pilih paket yang diinginkan
4. Lakukan pembayaran

---

## Technical

### API timeout, apa yang harus dilakukan?

Beberapa langkah yang bisa dicoba:

1. Cek koneksi internet
2. Pastikan API key valid
3. Cek status server di [status.qouta.id](https://status.qouta.id)
4. Jika masalah berlanjut, hubungi support

### Bagaimana cara melaporkan bug?

Buat issue di GitHub repository atau hubungi tim support melalui email di support@qouta.id.
